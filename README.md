## Installation
Github:

    git clone https://github.com/I2rys/DisPwned
    
NPM Packages:
```
npm i discord.js && npm i puppeteer && npm i is2
```

## Setup
1. Open index.js
2. Find **Self** > **bot_token** put your Discord bot token there.

## Usage

    node index.js
